Домашнее задание №2, вариант 18.

Предметная область:
- Справочник: Кафедры
- Основная таблица: Преподаватели
- Числовое поле: publications

Что входит в проект:
- Chair.cs
- Teacher.cs
- DatabaseManager.cs
- ReportBuilder.cs
- Program.cs
- chair.csv
- teacher.csv
- 3 схемы PlantUML
- Homework2_Variant18_CSharp.csproj

Функциональность:
1. Просмотр всех кафедр.
2. Просмотр всех преподавателей.
3. Добавление преподавателя.
4. Редактирование преподавателя.
5. Удаление преподавателя.
6. Отчёты:
   - преподаватели по кафедрам;
   - количество преподавателей по кафедрам;
   - среднее число публикаций по кафедрам.
7. Фильтр преподавателей по кафедре.
8. Экспорт таблиц в CSV.
9. Дополнительно из методички:
   - Numbered()
   - Footer(...)
   - SaveToFile(...)

Особенности реализации:
- Используется SQLite.
- При первом запуске создаётся база teachers.db.
- Данные автоматически импортируются из chair.csv и teacher.csv.
- CSV-файлы копируются в выходную папку проекта автоматически через .csproj.
- В SQLite включены внешние ключи для каждого соединения.

Как запустить:
1. Открыть папку проекта в Visual Studio / Rider / VS Code.
2. Выполнить восстановление пакетов NuGet.
3. Запустить проект.
4. Если раньше уже была создана пустая база, удалите файл teachers.db и запустите проект снова.

Примечание:
- Отчёты можно не только вывести на экран, но и сохранить в текстовый файл.
- После экспорта создаются файлы chair_export.csv и teacher_export.csv.
